# 航空票价动态监测与分析看板

公开访问版航空票价研究看板。网站使用脱敏聚合数据，不连接私人数据库，也不包含爬虫、Cookie、账号、原始采集记录或运行日志。

## 在线功能

- 自定义出行开始、结束日期
- 航线筛选
- 最大值、最小值、平均值、中位数、25分位、75分位切换
- 提前3天、提前7天、全部可用提前期平均
- 指定出行日历史价格追踪
- 与顶部日期范围联动的监测批次完整度

## 数据安全边界

仓库只保存以下聚合结果：

- 航线—出行日期—购票提前期价格统计
- 航线—出行日期—监测日期价格统计
- 航班维度的最低价、中位价、最高价和样本量
- 每日采集批次数量

不会提交 SQLite 数据库及任何原始采集环境信息。

## 本地预览

```powershell
python -m http.server 4174 --bind 127.0.0.1
```

访问 `http://127.0.0.1:4174/`。

## 从私人数据库更新公开数据

数据库只在本地读取，不会被复制：

```powershell
python scripts/export_public_data.py --db "D:\path\to\flight_prices.db"
```

重新提交生成的 `public/data/dashboard_data.json` 后，Vercel会自动部署新数据。
